软件名称：Inno Setup
版本：4.0.9
主页：http://www.jrsoftware.org/
运行环境：Win95/98/Me/NT/2000/XP


软件介绍：
      Inno Setup是一个免费的安装制作软件，它既有
  适合新手的脚本向导，又有良好的脚本编辑环境，能
  够快速制作出标准Windows2000风格的安装界面，足以
  完成一般的安装任务，在新版本版中软件加入了对多语
  言的支持，这个功能的实现相较于NSIS来得简单多了，
  推荐大家使用新的版本。在新版本中我自行按软件的关
  联程序注册的健值编写的注册表关联文件的编写例子，
  其中我还增加了一个源软件没有的“新建 Inno Setup 
  脚本  .iss”健值，希望该例子能对大家编写注册表脚
  本有所帮助，其实该软件也支持识别系统分别安装的不
  同的文件具体见例子吧(安装时注意选择是否使用安装
  软件关联注册表或使用软件本身关联注册表[二选一，
  如果全部安装也没问题，只是让大家试试这个功能罢了])！
  在这次的更新中我打包了一个专门替换Inno Setup
  生成的安装包的图标的软件(是官方网站推荐的)，只要
  大小不一样的图标替换时容易出现文件已损坏等字样，
  其实不要紧，有时虽然提示是坏的其实是可以用，当然
  也有时是真的损坏了，主要是备份就行了。

特别说明：
       在汉化本软件时因某些ASCII字符串长度不够，
  我就将这些字符串的快捷键去掉没有显示，望大家
  谅解。在安装目录中的Language文件夹中有生成安
  装文件的简体中文与繁体中文语言包。
  
在以下是各目录在脚本中相对应的代码。
{app}  应用程序目录
{pf}   Program Files 目录
{cf}   Common Files 目录
{win}  Windows 目录
{sys}  Windows system 目录   
{src}  安装程序源目录
{sd}   系统盘根目录
{commonstartup}  普通的启动文件夹
{userstartup}    用户启动文件夹
